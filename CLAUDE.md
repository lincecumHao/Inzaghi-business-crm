## Architecture

**Stack:** Next.js (App Router) + PostgreSQL via Supabase + Vercel deployment + Google APIs (Drive, Docs, Gmail)

**Auth:** Google OAuth via Supabase — no role/permission system, all authenticated users see the same data.

**Path alias:** `@/` maps to `./src/`

**React Compiler** is enabled (`next.config.ts` → `reactCompiler: true`). Do not manually memoize with `useMemo`/`useCallback` where the compiler handles it.

**Tailwind CSS v4** — uses PostCSS plugin, not the v3 config file approach.

## Domain Model

```
Customer
├── contacts[]           # multiple contacts per customer
└── orders[]
    ├── items[]          # line items with is_commissionable flag
    ├── status           # Pending → 已開報價單 → 已開發票 → 已付款
    ├── is_main_contract # only main contracts generate commissions
    ├── payment_date
    └── commission_schedule[]  # auto-generated on payment, 4 entries
```

Add-on contracts (`is_main_contract = false`) have no commission schedule.

## Commission Calculation Rules

Commission schedules are generated automatically when an order reaches 已付款 status and `payment_date` is set. The logic is pure and should be unit-tested exhaustively:

- **Disbursement dates:** 4 payments per year, each on the first day of the next quarter relative to `payment_date`
  - Example: payment 2026-07-03 → disbursements on 2026-10-01, 2027-01-01, 2027-04-01, 2027-07-01
- **Year boundaries:** counted from `payment_date`, not calendar year
  - Year 1: payment_date to payment_date+1yr−1day
  - Year 2: same pattern
  - Year 3+: same pattern
- **Rates:** 50% of commissionable item subtotal for years 1–2; 30% for year 3 onward

## Quote Number Format

Auto-generated by the system: `Q` + YY + MM + DD + 2-digit daily sequence (resets to 01 each day).
Example: `Q26032801`

## Google Drive Folder Structure

```

    └── {YYYY}/
        ├── 合約/
```

The system stores Google Drive links in the database rather than file contents.

## Document Generation

- **Quotes:** Generate a filled Excel file (user exports to PDF themselves — system does not produce PDF)
- **Contracts:** Clone a Google Docs template and fill variables: 甲方公司名稱, 甲方簽名人, 金額, 訂閱期間

## Reminder System

Per-customer reminder offset (N months before contract end). Each contract triggers at most one reminder email via Gmail API.
